مفهوم الخوارزمية — Algorithme
التعريف · الخصائص · البنية العامة
▾
تعريف

الخوارزمية هي مجموعة من التعليمات المرتبة والمحدودة التي تُنفَّذ خطوة بخطوة لحل مسألة ما أو الوصول إلى نتيجة معينة.
Algorithme : suite finie et ordonnée d'instructions permettant de résoudre un problème.

✅ خصائص الخوارزمية الجيدة
محدودية التعليمات · ترتيب منطقي · وضوح كل خطوة · الوصول إلى نتيجة

🔧 بنية الخوارزمية الأساسية:

Syntaxe — بنية الكتابة
Algorithme nom_algorithme ; Var nom_variable : type ; Début { instructions } Fin
الكلمة المحجوزةدورها
Algorithmeتعريف اسم الخوارزمية
Var أو Variableالتصريح عن المتغيرات
Débutبداية جزء التعليمات
Finنهاية الخوارزمية
⚠️ ملاحظة مهمة
يتم التصريح عن المتغيرات باستعمال الكلمة المحجوزة Var أو Variable — وذلك قبل الكلمة Début، وليس داخل التعليمات.

المتغيرات — Variables
التعريف · الأنواع · قواعد التسمية
▾
تعريف

المتغير (Variable) هو اسم يُعطى لموضع في الذاكرة يُخزَّن فيه قيمة قابلة للتغيير أثناء تنفيذ البرنامج.
Variable : emplacement mémoire repéré par un nom, pouvant contenir une valeur modifiable.

📋 أنواع البيانات الأساسية:

النوعمثال على القيمالاستعمال
Entier 5، -3، 100 الأعداد الصحيحة
Réel 3.14، -2.5، 6.3 الأعداد العشرية
Caractère 'A'، 'z'، '3' حرف واحد
Chaîne "Ahmed"، "Bonjour" سلسلة نصية
Booléen Vrai، Faux قيمة منطقية

📝 قواعد تسمية المتغيرات:

✅ قواعد صحيحة
— يبدأ الاسم بحرف (لا برقم)
— يحتوي على أحرف وأرقام وشرطة سفلية _
— لا يحتوي على مسافات
— لا يكون كلمة محجوزة مثل Début أو Fin
❌ أمثلة خاطئة
2Moy_Etu ← يبدأ برقم (خطأ!)   |   mon nom ← يحتوي مسافة (خطأ!)
مثال — Exemple
Algorithme Calcul_Moyenne ; Var Note1, Note2 : Réel ; NomEleve : Chaîne ; Age : Entier ; Début { ... } Fin

تعليمات الإدخال والإخراج
lire · ecrire · الإسناد
▾
إدخال — Lecture

lire(x) — تقرأ قيمة من لوحة المفاتيح وتُخزِّنها في المتغير x

إخراج — Écriture

ecrire(x) — تعرض قيمة المتغير x على الشاشة

🔑 تعليمة الإسناد ←
تُستعمل لإعطاء قيمة لمتغير:  A ← 20  أي: ضع القيمة 20 في المتغير A
مثال شامل — Exemple complet
Algorithme Saisie_Affichage ; Var S : Chaîne ; A, B, C : Entier ; Début ecrire('entrez S') ; { عرض رسالة } lire(S) ; { قراءة قيمة S } A ← 20 ; B ← A + 3 ; C ← A + B ; ecrire(A, B, C) ; { عرض النتائج } Fin

الهيكل الشرطي — Si...Alors
Si · Sinon · بنية القرار
▾
تعريف

الهيكل الشرطي يسمح بتنفيذ تعليمات مختلفة حسب صحة أو خطأ شرط معين.
Structure conditionnelle : permet d'exécuter des instructions différentes selon qu'une condition est vraie ou fausse.

شرط بسيط
Si condition Alors instruction(s) Fin Si
شرط مع حالة بديلة
Si condition Alors instruction(s) Sinon instruction(s) Fin Si

🧪 مثال تطبيقي — قراءة قيمة PH:

Algorithme Analyse_PH ; Var PH : Réel ; Début ecrire('entrez la valeur de PH : ') ; lire(PH) ; Si PH < 7 Alors ecrire('حامضي') ; Sinon Si PH > 7 Alors ecrire('قاعدي') ; Sinon ecrire('محايد') ; Fin Si Fin Si Fin
٥
هيكل التكرار — Pour...Faire
التكرار المحدد
▾
تعريف

هيكل التكرار Pour...Faire يُنفِّذ مجموعة من التعليمات عدداً محدداً من المرات.
Pour...Faire : répète un bloc d'instructions un nombre de fois déterminé à l'avance.

البنية العامة
Pour compteur ← valeur_debut à valeur_fin Faire instruction(s) ; Fin Pour

🧪 مثال — حساب معدل n تلميذ:

Algorithme Calcul_Moyennes ; Var i, n, x, y, z : Entier ; moy : Réel ; Début ecrire('entrer le nombre des élèves') ; lire(n) ; Pour i ← 1 à n Faire ecrire('entrer les notes') ; lire(x, y, z) ; moy ← (x + y + z) / 3 ; ecrire(moy) ; Fin Pour Fin
💡 متى نستخدم Pour...Faire؟
عندما نعرف مسبقاً عدد مرات التكرار — مثل: معالجة n تلميذ، حساب جدول ضرب، طباعة سلسلة من الأرقام.

الأشكال · AlgoBox · تحليل المسألة
▾
تعريف

المخطط الانسيابي (Organigramme) هو تمثيل بياني يساعدنا في حل المسائل والمشكلات عبر تسريع العمليات الحسابية ومن ثم القيام بإظهار النتائج — يبدأ بـ DEBUT وينتهي بـ FIN.
Organigramme : représentation graphique d'un algorithme à l'aide de symboles normalisés.

💻 برنامج AlgoBox
البرنامج الذي يُستعمل في إنشاء المخططات الانسيابية يُدعى AlgoBox. في هذا البرنامج توجد الثوابت والمتغيرات فقط ولا توجد Var.
⚠️ في المخططات الانسيابية
— توجد الحلقة Pour و Tant que في خرائط القرار/التفرع
— يتم الوصول بين الأشكال بإستعمال سهم يُسمى اتجاه التدفق (السريان)

🔑 الكلمات المحجوزة في الخوارزمية
الخوارزمية هي عبارة عن كلمات محجوزة. تُستعمل الكلمة المربعة للثوابت والمتغيرات. يمكن لاسم المعرّف أن يحتوي على رموز حرفية وعددية والرمز (_) وعلى فراغات.

❌ تصحيح
لا يمكن لاسم المعرّف أن يحتوي على فراغات — هذه العبارة خاطئة، التصحيح: لا يجوز وجود فراغات في أسماء المتغيرات.

النوع المنطقي — Booléen

المتغير من نوع Booléen لا يأخذ إلا قيمتين فقط: Vrai (صحيح) أو Faux (خاطئ). ويُستعمل غالباً مع عمليات المقارنة مثل: A > B أو A = B.

🔢 مثال تتبع — خوارزمية Affectation:

Algorithme affectation ; Var A, B : Entier ; C : Booléen ; con D ← 3 ; Début A ← D * 2 ; { A = 6 } B ← D - 1 ; { B = 2 } C ← A > B ; { C = Vrai (6>2) } A ← B ^ D ; { A = 2^3 = 8 } B ← D - A ; { B = 3-8 = -5 } C ← A < 0 OU B > 0 ; { C = Faux OU Faux = Faux } Fin
